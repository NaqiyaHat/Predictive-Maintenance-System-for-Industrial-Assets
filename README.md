# Predictive Maintenance System for Industrial Assets

An end-to-end predictive maintenance pipeline designed to analyse industrial sensor data and detect early signs of equipment failure using time-series data.

## Overview
This project focuses on analysing multivariate time-series data from industrial systems (e.g., bearing temperature readings) to identify abnormal patterns and potential failures before they occur.

The system simulates real-world predictive maintenance workflows used in industrial environments such as SCADA systems and IoT-enabled machinery.

## Key Features
- Processing and analysis of large-scale time-series sensor data  
- Detection of abnormal patterns and potential system faults  
- Data-driven insights for predictive maintenance decision-making  
- Simulation of real-world industrial monitoring workflows  

## Dataset
- Bearing temperature data collected over a 6-month period  
- Recorded at 10-minute intervals  
- Represents realistic industrial equipment monitoring scenarios  

## Tech Stack
- Python  
- Pandas (data processing)  
- NumPy (numerical analysis)  

## Project Structure
- `bearing_temp_project.py` – main analysis script  
- `bearing_temp_6months_10min.csv` – dataset used for modelling  
- `README.md` – project documentation  

## How It Works
1. Load and preprocess time-series sensor data  
2. Analyse trends and variations in temperature readings  
3. Identify anomalies or unusual behaviour patterns  
4. Use insights to simulate predictive maintenance decisions  

## Use Case
This system demonstrates how industrial companies can:
- Monitor equipment health in real time  
- Detect faults early and reduce downtime  
- Improve operational efficiency through data-driven maintenance
